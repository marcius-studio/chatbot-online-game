* [x] Vue.js 2.x
* [x] Vue Apollo

**Server-side**

## Installation

Required [Nodejs LTS](https://nodejs.org/en/) 

```node
$ cd client && yarn
$ cd server && yarn
```

### Run

```node
$ cd client && yarn run serve
$ cd server && yarn run start
```

### Build

```node
$ cd client && yarn run build
$ cd server && yarn run build
```
